Решение биматричных игр размерности 2х2 в смешанных стратегиях:

Обозначим стратегии игроков символами:
\begin{itemize}
\item Первого игрока P = (p; 1 - p)
\item Второго игрока Q = (q; 1 - q)
\end{itemize}

Функция выигрыша игрока A:
$$
F_A (p, q) = p[qa_11 + (1-q)a_12] + (1-p)[qa_21 + (1-q)a_22]
F_B (p, q) = p[qb_11 + (1-q)b_21] + (1-p)[qb_12 + (1-q)a_22]
$$

$\displaystyle F_{A}( p,q) \ =\ 1\ -q\ +\ p\ ( 3q\ -\ 1)$

$\displaystyle p\ =\ 1$, если $\displaystyle 3q-1 >0$

$\displaystyle p=0$, если $\displaystyle 3q\ -\ 1< 0$
